    Описание программы

    абдукторы бедра

    Дополнительное оборудование: голеностопная манжета.

    Исходное положение: коленно-кистевая стойка, точка подвешивания вертикально над манжетой, голень разгружена.

    Движение: отведение согнутой ноги с кратковременной задержкой в положении максимального отведения

    Рекомендации: исключить разворот таза. При отведении голень параллельна полу.
